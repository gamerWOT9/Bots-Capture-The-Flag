Régles principales

1. Le jeu est composé d'un terrain de jeu avec deux bases opposées.
2. Chaque équipe possède un drapeau et doit le protéger de l'autre équipe.
3. L'objectif du jeu est de capturer le drapeau de l'adversaire et de le ramener à votre base.
4. Les robots peuvent se déplacer librement sur le terrain de jeu et interagir avec les autres robots et objets.
5. Les robots peuvent communiquer entre eux afin de coopérer pour atteindre l'objectif du jeu.
6. Les robots peuvent être équipés de différents types d'armes et de munitions pour détruire les autres robots.
7. Les robots peuvent être équipés de différents types de capteurs pour détecter les autres robots et les objets.

Alors

Comment nous avons conçu notre robot pour le jeu de capture de drapeau:
Le premier jour nous avons étudié les règles du jeu et nous avons essayé de comprendre comment les autres robots fonctionnaient. Le deuxième jour, nous avons commencé à concevoir notre propre robot. Nous avons d'abord dessiné un schéma de notre robot, puis nous avons commencé à assembler les différentes pièces. Le troisième jour, nous avons terminé l'assemblage de notre robot et nous avons commencé à le tester. Nous avons ajusté certains de ses composants afin qu'il puisse mieux fonctionner. Le quatrième jour, nous avons continué à tester notre robot et nous avons essayé de l'améliorer encore. Nous avons également essayé de le faire fonctionner avec d'autres robots. Le cinquième jour, nous avons terminé de tester notre robot et nous l'avons encore amélioré. Nous avons ensuite préparé notre robot pour la compétition.
